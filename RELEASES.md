# Releases

버전별 주요 변경 요약. 상세 개발 이력은 로컬 `CHANGELOG.md`, 배포 시점은 git 태그(`vX.Y.Z`)로 관리.

## v1.1.0 — 2026-07-21

- **여러 LLM 연결** — 로컬 CLI(Claude·Codex) + API(OpenAI·Anthropic·Gemini·OpenRouter·Groq·xAI·Ollama·LM Studio·커스텀). API 키는 이 PC에만 암호화 저장하고 레포엔 올라가지 않음.
- **대화 개선** — 멀티턴 문맥 유지, 지난 대화는 설정 "대화내역" 탭에서 확인·초기화, 말풍선은 잠시 뒤 자동으로 닫힘(설정에서 시간 조절, 0이면 안 닫힘).
- **쿠폰 시스템** — 코드별 영구/1회용·보상(해금·XP)·만료를 다루는 구조. 코드는 소스에 해시로만 저장.
- **클릭 영역 정리** — 오리 실루엣에 맞게 클릭 판정을 좁혀 빈 귀퉁이는 바탕화면으로 통과.
