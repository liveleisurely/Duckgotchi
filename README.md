# 말랑오리

개발하다 막힐 때 만지작거리는 데스크탑 오리. 항상 화면 위에 떠 있고(전체 모니터를 넘나듦), 드래그해서 던지거나 더블클릭해서 다양한 반응을 보게 하거나, 막힌 문제를 오리한테 말하면 실제 Claude/Codex가 답해준다. 질문하고 놀수록 XP가 쌓여서 오리의 머리·부리·눈·날개·다리·발·꼬리 등 14개 파츠를 원하는 모양·색으로 하나씩 커스터마이징할 수 있다.

## 실행

```
run.bat 더블클릭
```
(처음 실행이면 `npm install` 자동으로 됨. 이후엔 바로 뜸)

또는:
```
npm install   # 최초 1회
npm start
```

## 조작법

| 동작 | 결과 |
|---|---|
| 짧게 클릭 | 오리 위에 말풍선이 뜨고 채팅 시작 |
| 더블클릭 | hop·스핀·바운스·젤리·하트비트 등 랜덤 반응 + 효과음 (14종 중 랜덤, 직전과 안 겹침) |
| 드래그 | 오리를 던짐 (관성 물리로 미끄러지다 화면 끝에서 튕김) |
| **꾹 누르고 0.4초 정지 후** 드래그 | 그 자리에서 제자리 회전 (이동과 안 섞임) |
| 우클릭 | 설정 패널 (AI 제공자 선택, 프롬프트 편집, 파츠 커스터마이징) |
| 좌상단 ✨ XP 배지 클릭 | 커스터마이징 탭으로 바로 이동 |
| 우상단 🌙 | 오리 재우기 (종료) |
| `Ctrl+Shift+Q` | 어디서든 종료 |

## 채팅

- 오리 클릭 → 말풍선에 막힌 문제를 적고 Enter
- 우클릭 설정에서 **Claude** 또는 **Codex** 중 선택 가능 (각자 컴퓨터에 설치·로그인된 CLI를 그대로 씀 — 별도 API 키·과금 없음)
- 오리한테 줄 지침(프롬프트)도 설정에서 자유롭게 수정 가능
- 같은 대화 중엔 문맥이 이어짐 (세션 유지)
- 답변에 관련 링크가 있으면 오리 머리 위로 구름 모양 말풍선이 따로 떠서 보여줌

## XP·커스터마이징

- 질문 1번 = +1 XP, 더블클릭 반응 1번 = +1 XP
- 우클릭 → 커스텀 탭에서 원하는 파츠(머리/몸통/부리/왼눈/오른눈/왼날개/오른날개/왼다리/오른다리/왼발/오른발/꼬리/볼터치/머리장식)에 XP를 투자해서 레벨업
- 레벨업할 때마다 그 파츠의 모양(구·상자·원뿔·원통·팔면체·고리·캡슐)과 색 옵션이 하나씩 늘어남 — 한번 푼 옵션은 계속 팔레트에 남아서 자유롭게 조합 가능
- 볼터치·머리장식은 기본 상태에선 아예 안 보이다가, 처음 레벨업하는 순간 짠 하고 나타남

## 구성

```
duck-widget/
├── main.js       Electron 메인 프로세스 (창 생성, Claude/Codex 호출, IPC)
├── preload.js    contextBridge로 duckAPI 노출
├── index.html    렌더러 — Three.js 오리, 물리, 채팅/구름 말풍선, 설정·커스텀 패널
├── run.bat       더블클릭 실행용
└── CHANGELOG.md  세션 중 오간 요청과 대응을 시간순으로 정리한 기록
```

## 참고

- 오리 모양은 전부 코드로 조립한 단순 3D 도형(구체 등)이고 외부 3D 에셋을 쓰지 않음
- `assets/duck/`, `assets/duck/CREDITS.txt`는 이전 시도(실사 스캔 모델)의 흔적으로, 현재 렌더링에는 사용되지 않음 — 지워도 무방
- 자세한 변경 이력은 `CHANGELOG.md` 참고
